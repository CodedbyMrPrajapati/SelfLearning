# RL/WIDS — Notebooks and Exercises

This folder contains Reinforcement Learning (RL) exercises and notebooks from the WIDS module inside the SelfLearning repository.

Contents
- A1A_TheGamblersRuin.ipynb — Jupyter notebook exploring the Gambler's Ruin problem.
- A1B_Building_Modular_Games/ — Exercises and code for building modular games (directory).
- A3.ipynb — Notebook for assignment A3.
- Week2.ipynb — Week 2 notebook for WIDS/RL.

Purpose
This README documents the purpose of the RL/WIDS folder and provides instructions to run and view the notebooks locally.

Requirements
- Python 3.8+ recommended
- JupyterLab or Jupyter Notebook
- Common data science packages: numpy, pandas, matplotlib, seaborn
- RL-related packages (if used in notebooks): gym, stable-baselines3 (install only if required by a notebook)
